# React

[英文官网:](https://reactjs.org/)  
[中文官网:](https://react.docschina.org/)

## 是什么

1. 用于动态构建用户界面的 JavaScript 库(只关注于视图)
2. 由 Facebook 开源

## 特点

声明式编码  
组件化编码  
React Native 编写原生应用  
高效（优秀的 Diffing 算法）

## 高效的原因

使用虚拟(virtual)DOM, 不总是直接操作页面真实 DOM。  
DOM Diffing 算法, 最小化页面重绘。
